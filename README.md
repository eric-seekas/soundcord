#SoundCord

A phonetic algorithm to make comparisons by phonetically similar terms easier.

##Installation
Add it to your Gemfile:
`gem install soundcord`

##Usage
###Basics


```ruby
"Jo√£o".phonetize # "J"
"Maria".phonetize # "MR"
"Helena".phonetize # "LM"
"Walmir".phonetize # "VLM"
```
